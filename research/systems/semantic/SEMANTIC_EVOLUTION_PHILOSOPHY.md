# DETECTING SILENT DRIFT

Symptoms include:

---

## Divergent Answers

Different operators answer different questions.

Question:

```text
What does health_score mean?
```

Different answers indicate:

```text
Reality
≠
Shared Meaning
```

---

## Adjective Explosion

Examples:

```python
health_score
overall_health_score
health_score_v2
performance_health_score
```

This often indicates that reality outgrew the original concept.

---

## Consumer Divergence

Different consumers assign different interpretations.

Examples:

```text
Dashboard
Risk Manager
Telemetry
Opportunity Agent
```

all consume:

```python
edge_score
```

but use it differently.

---

## Meaning Latency

Question:

```text
What question does this concept answer?
```

If explanation requires long discussions, semantic health may already be degrading.

Fast answers indicate healthy semantics.

High semantic latency is a warning signal.
